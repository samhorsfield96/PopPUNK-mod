# Overview of helper scripts

### analyse_pangenome.py

Using output of [WTBCluster](https://github.com/samhorsfield96/WTBcluster), calculate pangenome statistics for use with PopPUNK-mod.

### compare_distributions.py

Compares two pairwise distance distributions, calculating an assortment of statistics.

### compare_gridsearch.py

Compare statistics of pairwise distance distributions for a gridsearch of [Pansim](https://github.com/bacpop/Pansim).

### count_genes.py

Calculates statistics from pangenome presence/absence matrix

### create_ppmod_input.py

Point towards directory of output from `analyse_pangenome.py` and distances from [PopPUNK](https://github.com/bacpop/PopPUNK), match based on file prefix to generate an output that can be iterated through using PopPUNK-mod.

### distance_to_gridsearch.py

Compare query 2D pairwise distance distribution to a gridsearch to find the closest match.

### entrez_download.py

Download genomes using accession numbers from NCBI

### extract_poppunk_distances.py

Extract tab-separated file of distances from pkl and npy files generated by [PopPUNK](https://github.com/bacpop/PopPUNK)

### fit_nn.py

Experimental script for fitting a neural network to summary statistcs, mimicking Approximate Bayesian Computation

### get_core_mu.py

Calculates an adjusted core mutation rate using Jukes-Cantor subsitution mode.

### get_file_matches.py

Matches sample IDs in a large nested directory.

### get_metatraits.py

Use NCBI accession numbers to pull [metaTraits](https://metatraits.embl.de) information

### KS_distance.py

Experimental script for calculate Kolmogorov-Smirnov (KS) distribution distances for pairwise distance distributions.
